## Clone the repository

```sh
git clone https://github.com/growthbeing/laslesvpn-landing-page-assignment && cd laslesvpn-landing-page-assignment
```

## For Offline Development :man_technologist:

**For npm**

```sh
npm install 
npm run build 
npm run start
```

**For yarn**

```sh
yarn install 
yarn build 
yarn start
```
